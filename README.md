# Coin_Flip

Steps to run this Program on Android:

1.Download & Install Termux from play store
2.Open Termux
3.Type the Following One by One:
    pkg install git -y
    pkg install python3 -y
    Git Clone https://github.com/AbdulShan/Coin_Flip
    cd Coin_Flip
    python Coin_Flip.py
5.Have fun!!
